# iluvcoffee

## Description

[iluvcoffee](https://github.com/cpmachado/iluvcoffee) is the tutorial application in the nestjs
official course.

This being NestJS Fundamentals Course.

Refference for courses: <https://courses.nestjs.com/>

This is my work through it.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
